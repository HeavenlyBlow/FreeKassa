Модули «DualConnector» (DC) размещаются в «Global Assembly Cache (GAC)» операционной 
системы. При интеграции кассового ПО с DC не допускается:
 Поиск библиотек в коде по абсолютному или относительному пути;
 “Привязка” кода к определенной версии DC

Файлы находящие в DLL нужны исключительно для тестов!
